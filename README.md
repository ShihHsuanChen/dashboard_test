# Dashboard\_test

Test some techniques

- websocket
- flask_socketio
- render_template
- plotly
- moment.js

## Source Client
### Install
```sh
$ cd source_client
$ pip install -r requirements.txt
```
### Configure
Set the absolute path of sqlite file to ``DB_PATH`` in ``db/__init__.py``
### Run
```sh
$ python main.py
```

## Web Server
### Install
```sh
$ cd web_server
$ pip install -r requirements.txt
```
### Run
```sh
$ python main.py
```
Open [link](http://127.0.0.1:5003) at your browser
